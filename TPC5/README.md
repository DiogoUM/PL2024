## Descrição
Criar em python um simulador de uma máquina de vending.
Existe um ficheiro onde está guardada uma lista de produtos como por exemplo:

| id | Nome | Preço |
|----|------|-------|
|1   | água | 50c   |
|2   | bolo | 60c   |
...


Comandos como exemplos:

>**>>** LISTAR  
>>1 água 50 \
2c bolo 60c

>**>>** MOEDA 1e,10c,20c  
>> saldo = 1e,30c

>**>>** SELECIONAR 2 
>> saldo = 70c

>**>>** SAIR  
>> Troco 50c,20c
